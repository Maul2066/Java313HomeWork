7.	Напишите запрос, который выводит средний балл по экзаменам Васильевой.

USE db_9;

SELECT AVG(Ball) AS "Средний балл"
FROM exzamen, student
WHERE FIO REGEXP 'Васильева'

10.	Вывести список фамилий студентов получивших 5 баллов по дифференциальным уравнениям.

USE db_9;

SELECT FIO, Predmet, Ball
FROM student, exzamen
WHERE Predmet REGEXP 'Дифференциальные уравнения' AND Ball = 5

11.	Напишите запрос, который покажет,  сколько экзаменов  сдавал Шутов.

USE db_9;

SELECT FIO, COUNT(Predmet)
FROM Exzamen, Student
WHERE FIO REGEXP 'Шутов'
GROUP BY FIO;

12.	Выведите название кафедры студентов, не сдавших хотя бы один экзамен.

USE db_9;

SELECT Kafedra, COUNT(FIO)
FROM Kafedra, exzamen, Student
WHERE Ball IS NULL
GROUP BY Kafedra;

13.	Подсчитать количество кафедр на каждом факультете.

USE db_9;

14.	Подсчитать количество пятерок на физико-математическом факультете.

USE db_9;

SELECT Decanat, COUNT(Ball)
FROM Exzamen, Kafedra
WHERE Decanat REGEXP 'Физико-математический' AND Ball = 5
GROUP BY Decanat;
